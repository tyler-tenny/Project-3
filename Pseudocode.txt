FUNCTIONS:
	DisplayMenu
		- Show menu and prompt user for choice
			- Options: GetMatrices, AddMatrices AND PrintResults, MultiplyMatrices AND PrintResults
	GetMatrices
		- Get user input for matrix1 and matrix2
	AddMatrices
		- Add matrix1 and matrix2, store results in matrix3
	MultiplyMatrices
		- Multiply matrix1 and matrix2, store results in matrix3
	PrintResults
		- Print matrix1, operator symbol used, matrix2, '=', and matrix3

MAIN:
	GetMatrices
	DisplayMenu
	Prompt user to repeat program
